Tema 0 Ejercicio 2:Comparación entre innerText y innerHTML

Descripción

textContent establece o recupera solo el texto plano dentro de un elemento, sin interpretar las etiquetas HTML.
Esto significa que cualquier contenido HTML en la cadena se tratará como texto y se mostrará tal como está.
Se utiliza cuando deseas trabajar solo con el texto y no quieres que se interpreten las etiquetas HTML en el contenido.

innerHTML establece o recupera tanto el texto como el contenido HTML dentro de un elemento y permite que las
etiquetas HTML se interpreten y se representen como elementos HTML dentro del documento.
Se utiliza cuando deseas manipular el contenido HTML de un elemento, como agregar, eliminar o modificar
elementos HTML dentro de otro elemento.
