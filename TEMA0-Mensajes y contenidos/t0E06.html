Tema 0 Ejercicio 6:Uso de métodos del DOM

Descripción

En este ejemplo, se aplican los métodos createElement, removeChild y textContent

Se utiliza createElement para crear un nuevo elemento de lista (<li>)

Se utiliza el método removeChild para quitar el último elemento de la lista

Se utiliza textContent para el contenido de texto y dibujarlo en el DOM

En el cuadro de texto escriba un contenido

Se valida si hay contenido para agregar o elementos para remover
