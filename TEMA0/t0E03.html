Tema 0 Ejercicio 3:Mensajes con alert

Descripción

alert es una función que se utiliza para mostrar un mensaje emergente o una ventana de diálogo en el navegador web.
Este mensaje puede contener texto o información proporcionada como argumento a la función alert.
La función alert se utiliza para comunicar información simple o notificaciones importantes al usuario, como mensajes
de error, advertencias o confirmaciones.

Haga clic en cada uno de los botones para ver varias formas de manejar los mensajes de alerta.
